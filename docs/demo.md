# 라이브 데모

실제 동작하는 WZ-Grid를 직접 체험해보세요.

- **기본 기능**: 정렬, 필터, 체크박스, 컬럼 리사이즈, 드래그 재배치, 셀 편집
- **컬럼 타입**: text, number, date, boolean, select, badge, progress, link, button
- **페이징**: 100건 데이터 페이지네이션
- **그룹핑**: 부서별 그룹핑 & 소계
- **드래그 & 컨텍스트 메뉴**: 행 드래그 재배치, 우클릭 메뉴, 추가/삭제
